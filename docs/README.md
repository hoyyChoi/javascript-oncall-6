# javascript-oncall-6

**개발자 비상근무**

## 구현 과정

**목표** : 해당 달의 날짜의 특성들을 이용해 효율적인 비상근무표를 제작한다.

**세부 목표**

- 기능 먼저 구현 다 해놓기
- TDD 기반으로 코드 구현하기
- MVC 패턴 이용하기 (관심사를 분리하는 것)
- 함수(메소드) 길이 10라인 안넘어가게 작성하기

### 큰 프로세스

- 입력
  월과 시작 요일을 입력 (예외)
  평일 비상 근무 순번대로 사원들 입력
  휴일 비상 근무 순번대로 사원들 입력 (두개 다 입력 받고 예외)

비즈니스 로직 (고려 사항)

- 날짜 메소드 이용해서 날짜 구성하기 (입력 받으면 그때 생성) - 모델
- 휴일 따로 파일로 관리 -> 날짜 생기면 거기에 맞게 휴일 넣어준다.

- 베열 [평일 비상 근무 사람들 ]
- 배열 [휴일 비상 근무 사람들 ]
- 최종 날짜 배열[위 배열에서 하나씩 꺼내서 넣기]

  - 그리고 마지막꺼랑 비교하기(비교 자체는 바뀔 때만) 겹치면 그때 순서 변경

- 출력
  해당 월의 날짜 별로 출력!
  휴일은 휴일이라고 표시
